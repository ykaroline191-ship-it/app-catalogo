# Projeto Django — Catálogo

## Como rodar o projeto

Siga os passos abaixo para executar a aplicação em sua máquina:

1. Faça um **Fork** deste repositório para a sua conta do GitHub.

2. Clone o repositório do seu **Fork**:
   ```bash
   git clone https://github.com/SEU-USUARIO/catalogo.git
   ```

3. Acesse a pasta do projeto:
   ```bash
   cd app-catalogo
   ```

4. Inicialize o ambiente virtual:
   ```bash
   python -m venv venv
   ```

5. Ative o ambiente virtual:
   ```bash
   .\venv\Scripts\activate
   ```

6. Instale as dependências:
   ```bash
   pip install -r requirements.txt
   ```

7. Execute o servidor de desenvolvimento:
   ```bash
   python manage.py runserver
   ```
